
# **Chuỗi (Strings)**

**1.In chuỗi** 
```
o	Yêu cầu: Nhập chuỗi từ bàn phím và in ra màn hình.
o	Output mẫu: 
Nhap chuoi: hello
Chuoi: hello
```
**2.Tính độ dài chuỗi** 
```
o	Yêu cầu: Nhập chuỗi, tính độ dài bằng strlen hoặc thủ công.
o	Output mẫu: 
Nhap chuoi: hello
Do dai chuoi: 5
```
**3.Nối chuỗi** 
``` 
o	Yêu cầu: Nhập hai chuỗi, nối chúng bằng strcat và in kết quả.
o	Output mẫu: 
Nhap chuoi thu nhat: hello
Nhap chuoi thu hai: world
Chuoi sau khi noi: helloworld
```
**4.So sánh chuỗi** 
``` 
o	Yêu cầu: Nhập hai chuỗi, kiểm tra xem chúng có giống nhau không bằng strcmp.
o	Output mẫu: 
Nhap chuoi thu nhat: hello
Nhap chuoi thu hai: hello
Hai chuoi giong nhau.
```
**5.Sao chép chuỗi**
``` 
o	Yêu cầu: Nhập chuỗi, sao chép sang chuỗi khác bằng strcpy và in kết quả.
o	Output mẫu: 
Nhap chuoi: hello
Chuoi sao chep: hello
```
**6.Đảo ngược chuỗi**
``` 
o	Yêu cầu: Nhập chuỗi, đảo ngược thứ tự ký tự (ví dụ: "hello" → "olleh").
o	Output mẫu: 
Nhap chuoi: hello
Chuoi dao nguoc: olleh
```
**7.Chuyển đổi chữ hoa/thường** 
```
o	Yêu cầu: Nhập chuỗi, chuyển tất cả ký tự thành chữ hoa hoặc chữ thường.
o	Output mẫu: 
Nhap chuoi: HeLLo
Chuoi chu hoa: HELLO
Chuoi chu thuong: hello
```
**8.Đếm ký tự**
```
o	Yêu cầu: Nhập chuỗi và ký tự, đếm số lần xuất hiện của ký tự đó.
o	Output mẫu: 
Nhap chuoi: banana
Nhap ky tu: a
So lan xuat hien: 3
```
**9.Xóa ký tự** 
```
o	Yêu cầu: Nhập chuỗi và ký tự, xóa tất cả các ký tự đó khỏi chuỗi.
o	Output mẫu: 
Nhap chuoi: banana
Nhap ky tu: a
Chuoi sau khi xoa: bnn
```
**10.Kiểm tra chuỗi đối xứng**
``` 
o	Yêu cầu: Nhập chuỗi, kiểm tra xem chuỗi có phải là palindrome (đọc xuôi ngược giống nhau).
o	Output mẫu: 
Nhap chuoi: radar
Chuoi la palindrome.
```
**11.Tìm chuỗi con** 
```
o	Yêu cầu: Nhập chuỗi lớn và chuỗi con, tìm vị trí đầu tiên của chuỗi con trong chuỗi lớn (dùng strstr hoặc thủ công).
o	Output mẫu: 
Nhap chuoi lon: hello world
Nhap chuoi con: world
Vi tri: 6
```
**12.Thay thế chuỗi con** 
``` 
o	Yêu cầu: Nhập chuỗi lớn, chuỗi con cần thay, và chuỗi thay thế, thay tất cả chuỗi con bằng chuỗi thay thế.
o	Output mẫu: 
Nhap chuoi lon: hello world
Nhap chuoi con: world
Nhap chuoi thay the: everyone
Chuoi sau khi thay the: hello everyone
```
**13.Tách chuỗi** 
```  
o	Yêu cầu: Nhập chuỗi, tách thành các từ dựa trên dấu phân cách (ví dụ: khoảng trắng).
o	Output mẫu: 
Nhap chuoi: hello world everyone
Cac tu: 
1. hello
2. world
3. everyone
```
**14.Kiểm tra chuỗi hợp lệ** 
```  
o	Yêu cầu: Nhập chuỗi, kiểm tra xem chuỗi có thỏa mãn định dạng (ví dụ: email).
o	Output mẫu: 
Nhap chuoi: test@example.com
Chuoi la email hop le.
```
**15.Mã hóa chuỗi** 
```   
o	Yêu cầu: Nhập chuỗi và số dịch chuyển, mã hóa chuỗi bằng mã Caesar (dịch chuyển ký tự).
o	Output mẫu: 
Nhap chuoi: hello
Nhap so dich chuyen: 3
Chuoi ma hoa: khoor
```
